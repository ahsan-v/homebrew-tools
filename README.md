# ahsan-v/tools

My personal Homebrew tap.

## Install

```sh
brew tap ahsan-v/tools
```

Then install any formula from the tap by name.

## Formulae

| Name  | Description                                              | Source                              |
| ----- | -------------------------------------------------------- | ----------------------------------- |
| `kpf` | kubectl port-forward manager with stable local ports     | https://github.com/ahsan-v/kpf      |

## Usage

```sh
brew install kpf          # after tapping
brew install ahsan-v/tools/kpf   # one-shot without tapping first
brew upgrade kpf          # pull in new versions
```
